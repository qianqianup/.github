### 👋 关于我们：实干派的开源阵地

欢迎来到 **千千向上**。这里的名字源于最朴素的真理：“好好学习，天天向上”。

我们是一个由技术爱好者组成的开源实践组织。我们不讲空话，只看代码。在这里，在这个组织里，每一行代码都代表着一次向上的攀登。

---

### 📚 组织公约 (Code of Conduct)

加入千千向上，意味着你认可**“技术向上，品格向上”**的价值观。我们将参考文本中的软性约束转化为以下行动准则：

#### 1. 核心驱动：求知与热爱 💡
* **保持饥饿**：加入组织的前提是你热爱编程。无论你是技术大牛还是入门新手，**“想学”**是留在这里的唯一理由。
* **默默成就**：我们欢迎热烈的技术探讨，也同样尊重那些不善言辞、埋头写代码的实干家。行动永远比争论更有力量。

#### 2. 开源精神：分享与边界 ⚖️
* **纯粹分享**：我们提倡以技术交流为目的的开源，严禁利用组织名义或资源进行非法牟利。
* **版权意识**：
    * **自我保护**：学会使用 License（开源协议）捍卫自己的劳动成果。
    * **尊重他人**：严禁未经允许的抄袭、搬运或代码泄露。在他人的开源成果上学习时，请保留原作者的署名，这是一种基本的职业素养。

#### 3. 社区礼仪：尊重与谦逊 🤝
* **拒绝傲慢**：技术没有终点，不要因为一时的领先而诋毁或轻视其他开发者。
* **平视交流**：三十年河东，三十年河西。只要肯努力，曾经的初学者也会超越你。保持谦虚，以和谐为本。
* **有效沟通**：遇到 Bug 或建议时，请**针对问题**，而不是**针对人**。拒绝抬杠、无视和辱骂。

#### 4. 长期主义：积累与质变 📈
* **拒绝速成**：你的努力程度决定了你的技术高度。不要轻言放弃。
* **回望成就**：多年后当你回头俯瞰，你会发现，正是今天这些看似枯燥的代码提交，让你达到了曾经仰望的高度。
