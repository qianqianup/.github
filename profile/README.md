### 👋 关于我们

欢迎来到 **千千向上**。

我们是一个务实的开源技术组织。正如名字所示，我们的初衷非常简单：在技术的世界里，通过不断的学习与实践，每天进步一点点。

---

### 📚 组织公约

加入我们，代表你认可并愿意遵守以下共识：

#### 1. 核心驱动：求知与热爱 💡
* **学习第一**：加入组织的首要前提是热爱编程。无论你是技术大牛还是入门新手，愿意学习和分享是留在这里的唯一理由。
* **尊重实干**：我们欢迎热烈的技术探讨，也同样尊重那些不善言辞、默默提交代码的朋友。在千千向上，行动比语言更有力量。

#### 2. 开源底线：分享与版权 ⚖️
* **纯粹分享**：我们提倡以分享知识为目的，严禁利用组织的名义或资源进行非法牟利。
* **自我保护**：请学会使用开源协议（License）来保护你自己的原创作品。
* **尊重版权**：严禁未经允许的抄袭、洗稿或泄露他人隐私代码。在学习他人代码时，请保留原作者的署名，这是基本的职业素养。

#### 3. 社区礼仪：谦逊与沟通 🤝
* **拒绝傲慢**：技术没有终点，不要因为一时的领先而轻视其他开发者。保持谦虚，以和谐为本。
* **平等交流**：只要肯努力，昔日的初学者终将成长为大牛。我们平视每一位贡献者，不以资历论高低。
* **有效反馈**：遇到 Bug 或建议时，请针对问题本身进行建设性沟通。拒绝抬杠、无视和辱骂。

#### 4. 长期积累：拒绝浮躁 📈
* **坚持投入**：你的努力程度决定了你的技术高度。不要轻言放弃，也不要迷信速成。
* **见证成长**：多年后当你回头看，你会发现正是今天一点一滴的积累，让你达到了曾经仰望的高度。积极参与开源，帮助他人的同时，也是在成就自己。
